# Social Post Copy Generator

Generate platform-optimized titles, summaries, and hashtags for X, Dev.to, and Hashnode from a single product description.

## Features
- Multi-platform copy generation
- A/B title variants
- Platform-specific hashtags
- One-click copy
- No login required

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS
- TypeScript

## Development
```bash
npm install
npm run dev
```

## Deployment
Deployed on Vercel with custom domain via Cloudflare.
